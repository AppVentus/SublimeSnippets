SublimeSnippets
===============

This package provides some usefull snippets for sublimetext. Fell free to add yours !


PHP
---


'vd'+tab => var_dump();
'vde'+tab => var_dump();exit;
'pr'+tab => print_r();
'pre'+tab => print_r();exit;
'die'+tab => die();

TWIG
----

'trans'+tab => {{ 'your.key'|trans|desc('your.trad') }}
